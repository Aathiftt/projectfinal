## FFT Excel Formula Generator – Shake Table Analysis

This Streamlit web app generates step-by-step Excel formulas for FFT-based
analysis of shake table experimental data.

### Key Features
- Structured input for column mapping
- Excel-ready formulas (copy–paste)
- Clear paste instructions for each step
- Focused on FFT analysis (no generic Excel logic)

### Assumptions
- Time and corrected acceleration data are already available
- FFT is performed externally using Excel Data Analysis ToolPak
- FFT output is a single complex-valued column

### Tech Stack
- Python
- Streamlit
- GitHub + Streamlit Cloud
